
### Pro QR Code Generator 🧬
A modern, client-side QR Code Generator designed for GitHub Pages. Built with pure HTML, Tailwind CSS, and JavaScript. No server-side dependencies required.

### ✨ Features
Modern UI/UX: Sleek glassmorphism design with smooth animations and responsive layout.
Clipboard Integration: One-click "Paste" button to instantly grab URLs from the clipboard.
Advanced Customization:
Adjustable Foreground & Background colors.
Dynamic resizing (150px to 500px).
High Error Correction Level (H) to ensure scannability even if partially obscured.
Downloadable: Export generated QR codes instantly as high-quality PNG files.
Zero Dependencies: Runs entirely in the browser. No data is sent to any server.

### 📦 Installation & Deployment
To use this on your GitHub Page:
Create a Repository: Create a new repository on GitHub.
Upload File: Upload the index.html file provided above to the root of the repository.
Enable Pages:
Go to repository Settings > Pages.
Under "Source", select the main branch and click Save.
Access: Your site will be live at https://[your-username].github.io/[repo-name]/.

### 🛠️ Technology Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)
Styling: Tailwind CSS (CDN)
Logic: QRious.js (Library for QR generation)


## 📝 Usage
Paste a URL into the input field (or use the smart paste button).
Adjust colors and size using the advanced settings.
Click Generate QR Code.
Click Download PNG to save the image.
